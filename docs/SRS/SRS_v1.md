# Especificación de Requisitos de Software (v1.0)
## Funcionales
1. REQ-001: El sistema debe permitir login con usuario y contraseña.
2. REQ-002: El sistema debe permitir registrar la entrada de electrodomésticos.
3. REQ-003: El sistema debe generar un reporte de stock bajo.
4. REQ-004: El sistema debe permitir buscar productos por código SKU.
## No Funcionales
1. RNF-001: El tiempo de carga del dashboard no debe exceder 2 segundos.
2. RNF-002: Las contraseñas deben almacenarse encriptadas (SHA-256).5. REQ-007: El sistema debe permitir exportar inventario a PDF.
