# Sistema de Inventario - Importadora Sonib (Demo)
## Objetivo
Repositorio de práctica para Gestión de la Configuración de Software. Simula el control de versiones de la Importadora Sonib.
## Estructura
- /docs: Documentación de requisitos y diseño.
- /src: Código fuente (Python/JS).
- /config: Plantillas de configuración.
## Baseline
La versión v1.0 define el alcance inicial aprobado por el cliente.
